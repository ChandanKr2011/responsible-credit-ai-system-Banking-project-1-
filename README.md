# Responsible Credit Risk AI System

Balancing Predictive Performance, Fairness, and Explainability in Lending Decisions

## Problem Statement

Banks increasingly use AI systems to support lending decisions. However, predictive accuracy alone is insufficient in high-stakes domains like credit risk. This project explores how AI systems can be designed and evaluated not only for performance, but also for fairness, transparency, and policy impact.

## Objectives

This project aims to:

- Build credit risk prediction models
- Evaluate statistical and business performance
- Analyze fairness and potential bias
- Explain model decisions using interpretable AI techniques
- Simulate lending policy trade-offs

## Project Components

- Data preprocessing
- Credit risk modeling
- Model evaluation
- Explainability analysis
- AI governance & fairness assessment

## Tech Stack

- Python
- Pandas
- Scikit-learn
- SHAP
- Matplotlib
- Jupyter Notebook
